description: "This is how you create a welcome message (simple)"
---

<Info>
The system message "Send a random reply when a member joins" is required for this to work, can be in a private channel that only sigma bot sees.
</Info>

<Warning>
This requires `ADMINISTRATOR` Permission.
</Warning>


### Command

First of, use the `/set_welcome` command.

Fill out "Message" with the message you want for it.

Now send it and your done.

### Variable

Following Variable exist:

- [user]
