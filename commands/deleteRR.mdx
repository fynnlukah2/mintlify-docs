---
title: Delete Reaction Roles
description: "In this page, we'll show you how to use the reaction roles."
---

<Info>
**INFORMATION:** You will need Developer mode on. Check out how to turn it on at [discord dev mode](https://docs.sigmabot.lol/discord/dev-mode)
</Info>

To delete a reaction role, you will need to copy the message ID of the message where you want to delete it.

Firstly, run `/del_reaction_role <message_id> <emoji>`

Replace the "message_id" option with the id of the message where you want to remove the reaction role.

Replace the "emoji" option with the emoji of the reaction role you want to remove.

<Warning>
**This requires "MANAGE_ROLES" permsission.** 
</Warning>
